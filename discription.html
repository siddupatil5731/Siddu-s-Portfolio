Online Food Delivery

Modules and Functions

  • 1. Registration of Hotels and Users
    • 1.1 userRegistrationForm - Registers a new user by capturing their personal details.
    • 1.2 hotelRegistrationForm - Registers a new hotel by capturing its details and adding it to the system.
    • 1.3 loginUser - Authenticates and logs in a user to access the system.
    • 1.4 searchHotels - Allows users to search for hotels based on specific criteria.
    • 1.5 viewAllUsers - Displays a list of all registered users in the system.
    • 1.6 viewAllHotels - Displays a list of all registered hotels in the system.
    • 1.7 deleteHotel - Removes a hotel's registration from the system.
    • 1.8 editUserInformation - Allows users to update their registration details.
    • 1.9 editHotelInformation - Enables hotels to update their registration information.
  • 2. Food Varieties
    • 2.1 displayMenu - Displays the menu with all food items available in the selected hotel.
    • 2.2 placeOrder - Places a new food order for a user based on their selections.
    • 2.3 viewOrder - Shows the details of the current food order placed by a user.
    • 2.4 clearOrder - Cancels the current food order and clears its details.
    • 2.5 searchMenu - Searches for food items based on user input (e.g., name or category).
    • 2.6 displayReceipt - Generates and displays the receipt for a completed order.
    • 2.7 sortMenuByPrice - Sorts and displays menu items in ascending or descending order of price.
    • 2.8 displayTopItems - Displays the top-rated or most ordered food items.
    • 2.9 calculateAveragePrice - Computes and displays the average price of all food items on the menu.
  • 3. Shortest Distance and Efficient Delivery
    • 3.1 Manage Agents
      • 3.1.1 registerAgent - Registers a delivery agent and adds them to the system.
      • 3.1.2 loginAgent - Authenticates and logs in a delivery agent.
      • 3.1.3 updateAvailability - Updates an agent's availability status for delivery assignments.
      • 3.1.4 assignOrderToAgent - Assigns a new order to an available delivery agent.
      • 3.1.5 displayAgentList - Shows a list of all registered delivery agents.
      • 3.1.6 displayAgentDetails - Displays detailed information about a specific delivery agent.
      • 3.1.7 displayAgentOrderHistory - Displays the order history handled by a particular agent.
      • 3.1.8 displayAgentPerformanceStatistics - Shows performance metrics (e.g., success rate) of agents.
    • 3.2 Manage Orders
      • 3.2.1 displayOrderList - Displays all active and completed orders in the system.
      • 3.2.2 displayOrderDetails - Shows detailed information about a specific order.
      • 3.2.3 markOrderAsCompleted - Marks an order as completed and updates its status.
    • 3.3 Shortest Path
      • Calculates the shortest delivery route between two locations using an algorithm like Dijkstra's.
    • 3.4 Vehicle and Order Assignment
      • Calculates fuel consumption, delivery time, cost, distance, and route details for a specific order.
    • 3.5 Assign Vehicle to Order
      • Assigns the most suitable vehicle to a specific order based on delivery requirements.
  • 4. Feedback for Users and Hotels
    • 4.1 User Feedback
      • 4.1.1 Submit Feedback - Allows users to provide feedback on food or service.
      • 4.1.2 Display Vegetarian Feedback - Displays feedback specific to vegetarian food.
      • 4.1.3 Display Non-Vegetarian Feedback - Shows feedback specific to non-vegetarian food.
      • 4.1.4 Display Italian Food Feedback - Displays user feedback on Italian food items.
      • 4.1.5 Display French Food Feedback - Shows user feedback on French food items.
      • 4.1.6 Display Chinese Food Feedback - Displays feedback specific to Chinese food.
      • 4.1.7 Display All Feedbacks - Shows all feedback submitted by users.
      • 4.1.8 Display Feedback History - Displays the feedback history of a particular user.
      • 4.1.9 Display Ratings - Shows average ratings given by users for different foods or services.
      • 4.1.10 Display Feedback Statistics - Displays overall feedback statistics (e.g., average scores).
      • 4.1.11 Display User Names and Places - Lists users along with their feedback and locations.
    • 4.2 Hotel Feedback Menu
      • 4.2.1 Submit Hotel Feedback - Allows users to provide feedback specific to a hotel.
      • 4.2.2 Display All Hotel Feedbacks - Displays all feedback received for hotels.
      • 4.2.3 Display Hotel Feedback History - Shows historical feedback for a particular hotel.
      • 4.2.4 Display Hotel Ratings - Displays ratings given to various hotels.
      • 4.2.5 Display Feedback Analytics - Provides analytical insights based on hotel feedback.
    • 4.3 Display Both User and Hotel Feedbacks
      • It will display both feedback of Users and Hotels
